# Word Wiz

## Description
Word Wiz is a beginner Python project that analyzes a sentence entered by the user.

## Features
- Counts Spaces
- Counts Vowels
- Counts Letters 
- Counts Words

## Concepts Learned
- Variables
- User input
- String operations
- Basic problem solving

## Files
- `word_wiz.py`

## Learning Outcome
This project helped me practice working with strings and user input.
