# Contact Sphere

## Description

Contact Sphere is a Python program that helps you store and manage contacts through a simple menu-driven interface.

The program allows users to add, view, search, edit, and delete contacts, making contact management easy and organized.

## Features

- Add new contacts
- View all saved contacts
- Search contacts by name
- Edit contact names
- Update contact numbers
- Delete contacts
- Prevent duplicate contact names

## Concepts Used

- Variables
- Dictionaries
- Functions
- Loops
- Conditional Statements
- User Input
- CRUD Operations (Create, Read, Update, Delete)

## Files

- contact_sphere.py
